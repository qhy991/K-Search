## 关键学习

1. **公式正确性**: 必须使用 `d4_0 * (d_a * sumi - 8 * s_a)`
2. **共享内存**: 缓存激活量化避免重复计算
3. **DP4A**: INT8 点积加速的关键
4. **Warp 并行**: 充分利用 32 线程 warp
5. **统一架构**: 单一 kernel 处理所有批次大小
