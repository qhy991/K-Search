# W8A32C8 Q8_0 FP32 INT8 量化GEMM优化总结

## 任务概述

**算子类型**: 量化矩阵乘法 (Quantized GEMM)  
**模型**: DeepSeek V2 MoE Routing Up Projection  
**数据格式**: W8A32C8 (8-bit权重, 32-bit激活, Q8_0块量化)  
**矩阵维度**: N=5120, K=1536, M=variable (批大小)

## 量化格式说明

### Q8_0 块量化格式
每个数据块包含32个int8量化值，共享一个FP16缩放因子：
- 总大小: 34字节/块
- 缩放因子: 2字节 (FP16)
- 量化值: 32字节 (int8[32])

计算公式: `output = sum(activation[i] * weight_qs[i]) * scale`

### 关键优化技术

#### 1. 动态内核调度
根据批大小M选择最优内核：
```cuda
if (M <= 32) {
    // 内存受限: 使用循环展开内核
    q8_0_gemm_unrolled<1><<<grid, block>>>(...);
} else {
    // 计算受限: 使用标准内核
    q8_0_gemm_standard<1><<<grid, block>>>(...);
}
```

#### 2. K循环展开 (内存受限情况)
- 每次迭代处理2个K块
- 减少循环开销
- 提升指令级并行(ILP)

#### 3. 最优线程配置
- 线程块: 16×16 = 256线程
- N_PER_THREAD: 1 (每线程计算一个输出)
- Grid: 根据M和N动态计算

#### 4. 内存访问优化
- 合并全局内存访问
- 预计算激活行指针
- 完全展开的点积计算
