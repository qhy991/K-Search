# 模板定义文件

本目录包含通用的量化 GEMM 操作模板定义。这些文件用于：

1. **作为参考实现**：提供标准化的操作定义格式
2. **代码生成**：作为 AI 代码生成工具的输入模板
3. **测试基准**：提供标准尺寸的测试用例

## 模板特征

所有模板文件都包含以下特征：

- `"is_template": true` - 标识为模板文件
- `"tags": ["template:generic", ...]` - 包含模板标签
- `"description": "[TEMPLATE] ..."` - 描述前缀标识

## 与真实例子的区别

| 特征 | 模板文件 | 真实例子 |
|------|---------|---------|
| 位置 | `templates/` | 根目录或模型目录 |
| `is_template` | `true` | 不存在或 `false` |
| `model_architectures` | 通常缺失或通用 | 包含特定模型 |
| `test_configs` | 通常缺失 | 包含测试配置 |
| `performance_notes` | 通常缺失 | 包含性能说明 |
| `scenario` 标签 | 缺失 | 包含场景标识 |

## 使用建议

1. **开发新内核**：参考模板文件了解标准格式
2. **生成代码**：使用模板作为 AI 代码生成的输入
3. **测试验证**：使用模板进行通用测试

## 真实例子位置

真实模型的工作负载定义位于：

- 根目录：通用模型的标准尺寸定义
- `deepseek_v2/`：DeepSeek V2 模型特定定义
- `deepseek_v3/`：DeepSeek V3 模型特定定义
- `qwen2_5_7b/`：Qwen2.5-7B 模型特定定义

真实例子包含：
- 特定模型的尺寸配置
- 详细的测试配置（`test_configs`）
- 性能说明（`performance_notes`）
- 模型架构上下文信息
